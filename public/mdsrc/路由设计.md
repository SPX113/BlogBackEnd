# 路由设计



| 请求方法 |      请求路径      |   get参数    | post参数 |        备注        |
| :------: | :----------------: | :----------: | :------: | :----------------: |
|   get    |  /recommedArticle  |      /       |    /     |    获取推荐文章    |
|   get    |      /article      |      id      |    /     | 根据id获取文章信息 |
|   get    | /mdsrc/:article.md |      /       |    /     |    获取文章内容    |
|   get    |    /allarticle     |    pageNo    |    /     |  获取pageNo的数据  |
|   get    |      /comment      | name,comment |    /     |      提交评论      |
